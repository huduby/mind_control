# 🌱 마음 봄 (Ma-eum Bom)

AI를 활용한 감정 기록 및 감정 케어 웹앱

## 📖 프로젝트 소개

'마음 봄'은 사용자가 매일 자신의 감정을 한 문장으로 기록하고, AI가 감정을 분석하여 감정의 흐름을 이해하고 스스로를 돌아볼 수 있도록 돕는 감정 케어 웹앱입니다.

고등학생 자녀가 감정적으로 힘들어하는 모습을 보며, 자신의 감정을 안전하게 기록하고 객관적으로 돌아볼 수 있는 공간이 필요하다고 생각하여 개발하였습니다.

본 프로젝트는 LoRA 기반 감정 분류 모델과 OpenAI API를 활용하여 사용자의 감정 상태를 분석하고 공감 레터를 제공합니다.

---

## 🎯 주요 기능

### 1. 감정 기록

* 오늘의 감정을 한 문장으로 입력
* AI가 감정을 자동 분류
* emotion1(대분류), emotion2(세부 감정) 예측

### 2. 감정 데이터 저장

* Google Sheet와 연동
* 날짜, 감정 문장, 감정 분석 결과 저장

### 3. 마음 기후 분포

* 최근 30일간 감정 데이터를 분석
* 감정 비율 시각화

### 4. 공감 레터 생성

* OpenAI GPT API 활용
* 최근 감정 흐름을 기반으로 공감 메시지 생성
* 작은 실천 제안 제공

## ⚠️ 한계점

* 학습 데이터의 양이 충분하지 않아 감정 분류 정확도에 한계가 있음
* 감정 표현 방식이 다양하여 일부 문장은 정확하게 분류되지 않을 수 있음
* OpenAI API 사용량 제한에 따라 응답이 실패할 수 있음
